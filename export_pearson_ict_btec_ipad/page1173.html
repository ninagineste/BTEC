Data structures

Introduction

In this section, you will learn how to develop software programs that make use of data structures. There are many different types of data structures, including user-defined data types and record structures, and arrays. You will also learn how to use one-dimensional arrays.

Arrays

An array is a type of data structure which can store a large collection of data. You will create a software program, which will demonstrate how data can be stored and accessed in a one dimensional array.

Basic string handling

A string is a piece of text containing one or more characters (letters, digits, punctuation, etc.).
